Exercise physiology metrics analysis.

For this project, I used Python on Kaggle to analyze exercise physiology datasets. Kaggle is a data scientist platform where individuals collaborate and compete with each to solve problems and analyze datasets. I used Kaggle to analyze an exercise physiology dataset, which contained information such as VO2 (oxygen consumption), VCO2 (carbon dioxide production), FECO2 (fraction of expired CO2), RER (respiratory exchange ratio), METs (metabolic equivalents), and heart rate. My main goal was to analyze VO2, VCO2, and FECO2 and to visualize key inflection points: the gas exchange threshold (GET) and respiratory compensation point (RCP). These are important markers for exercise intensity and performance capacity.

I processed subject 1432’s raw data and then wrote a code to plot FECO2 and VCO2 over VO2. A scatter plot was used because it shows how one variable changes in relation to another. Scatter plots also make it easy to identify any trends which is crucial for determining GET and RCP thresholds. A vertical line was used to indicate GET and RCP on the plot. This project helped me apply my knowledge of exercise physiology in a practical, hands-on way while strengthening my skills in Python data visualization. It was rewarding to see how programming could make complex physiological concepts easier to understand and communicate visually.
