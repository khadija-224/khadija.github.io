Creative Coding and Game Deisgn

For this project, I created an interactive maze game using Scratch. My goal was to design a fun and engaging game that challenged players to complete a maze within a time limit. I customized my own sprites and added sound effects to make the game more personal and dynamic. To increase difficulty, I included multiple levels and made the game cheat-resistant by adding boundaries that reset the player if they tried to cut corners. This was my first time building a game, and it really helped me understand how to break down problems, think logically, and design with the user in mind. It also showed me that coding can be both creative and rewarding.

What is Scratch?

Scratch is a beginner-friendly coding platform by MIT that uses blocks instead of traditional code. It’s a great way to learn how programming works because you can snap together blocks to control characters, create games, and tell stories. It’s designed for all ages, especially those new to coding, and makes it easy to turn ideas into interactive projects.
